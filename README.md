# esp-idf-clion
configuration and setup clion with esp-idf

## Install and setup esp idf
1. Create directory esp
```
mkdir -p ~/esp
cd ~/esp
git clone --recursive https://github.com/espressif/esp-idf.git
```

2. change directory to esp-idf
```
cd ~/esp/esp-idf
```

3. run script install.sh
```
./install.sh
```
